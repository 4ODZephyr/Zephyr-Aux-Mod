
# Callout Reference Card Addon

Adds a Zephyr callout reference card item which provides quick reference information for in-game communication and callouts.

## Usage

The reference card can be added to a unit's inventory via editor, arsenal, or script:

```sqf
_unit addItem "Zephyr_Callout_Reference_Card";
```

## Requirements

- Zephyr Core addon
- Zephyr Unit Patch addon
